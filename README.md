# Gnanesh Portfolio

My personal portfolio website showcasing projects, skills, and experience.

## Description
This is my professional portfolio website built with HTML, CSS, and JavaScript. It displays my work, technical skills, and ways to get in touch.

## Tech Stack
- HTML5
- CSS3
- JavaScript
- Responsive Design

## Features
- Clean, modern design
- Project showcase
- Skills section
- About me
- Contact information
- Fully responsive
- Mobile-friendly

## Getting Started

### Prerequisites
- A modern web browser

### Installation
1. Clone the repository
   ```bash
   git clone https://github.com/gnanesh2310/Gnanesh-protfolio.git
   ```
2. Open `index.html` in your browser

## Usage
Simply open the portfolio website in any modern web browser to explore the projects and skills.

## Sections
- **Home** - Introduction and hero section
- **About** - About me and background
- **Projects** - Showcase of my work
- **Skills** - Technical skills and expertise
- **Contact** - Get in touch information
